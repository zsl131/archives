## · 开证明流程

+ 进入“证明模板管理”，找到相应模板后点“开证明”

+ 如果有身份证原件，则使用阅读器自动识别

+ 如果没有身份证原件，则输入身份证号搜索（前提是在村民表里有该人员）

+ 通过阅读器读取证件时需要将数据提交至村民表，如果不存在需要自动录入新村民
